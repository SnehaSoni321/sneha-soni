A polished, responsive portfolio website built with plain HTML, CSS, and JavaScript. It highlights projects, certifications, experience, and contact details in a modern one-page layout.

## ✨ What’s Included

- 🌙 Dark / light mode toggle with saved preference
- ⌨️ Animated typing effect in the hero section
- 📱 Fully responsive design for mobile, tablet, and desktop
- ✨ Scroll-reveal animations for cards and sections
- 🧭 Smooth navigation with a mobile-friendly menu
- 🧩 Sections for work, skills, projects, certifications, experience, about, and contact

## ▶️ Run Locally

```bash
git clone https://github.com/SnehaSoni321/sneha-soni.git
cd sneha-soni
start index.html
```

No installation or build step is required. Open the project in a browser to view it locally.
