## ✨ Features
 
- 🌙 **Dark / Light Mode** toggle
- ⌨️ **Animated Typing Effect** in hero section
- 🔲 **Floating Grid Background** (hero section only)
- 📱 **Fully Responsive** — works on mobile, tablet & desktop
- 🎞️ **Scroll Reveal Animations** on cards and sections
- ⚡ **Fast Loading** — no heavy frameworks or dependencies

## 🚀 Getting Started
 
### View Locally
 
```bash
# 1. Clone the repo
git clone https://github.com/SnehaSoni321/sneha-soni.git
 
# 2. Open in browser
cd sneha-soni
open index.html
```
 
No install, no setup, no server needed — just open `index.html` in your browser! ✅
